# SkillNotes

SkillNotes - веб-приложение Node.js для создания и хранения личных заметок.

Проект разработан в рамках итоговой дипломной работы курса Skillbox "Node.js".

## Описание функциональности

SkillNotes предоставляет следующие возможности:
- Регистрация пользователя
- Авторизация по логину и паролю
- Выход из личного аккаунта
- Создание заметок
- Просмотр заметок
- Редактирование заметок
- Архивирование заметок
- Восстановление заметок из архива
- Удаление отдельных заметок
- Удаление всех заметок из архива
- Скачивание заметок в формате PDF
- Фильтрация заметок по времени
- Поиск заметок по заголовку
- Постраничная загрузка заметок
- Поддержка Markdown

## Установка и начало работы

`git clone <URL_репозитория>` (клонировать репозиторий)

`cd 99_diploma` (перейти в папку с проектом)

`npm install` (установить зависимости)

При установке зависимостей автоматически выполняется установка браузера Chrome, необходимого для экспорта заметок в формате PDF. Если автоматическая установка завершилась ошибкой, необходимо выполнить установку Chrome вручную:

`npx puppeteer browsers install chrome`

## Настройка окружения

Создать файл `.env` в корне проекта и скопировать в него содержимое файла `.env-sample`. Поменять параметры подключения к базе данных:
- username
- password
- cluster

## Запуск проекта

Перед запуском приложение автоматически собирает клиентскую часть Svelte.

`npm run dev` (запуск в режиме разработчика)

`npm run start` (запуск в обычном режиме)

После запуска приложение будет доступно по адресу: `http://localhost:3000`.

## Подготовка базы данных

Приложение работает с базой данных MongoDB, поэтому не требует предварительного создания коллекций.

При первом запуске приложения автоматически выполняется:
- Подключение к базе данных
- Создание необходимых индексов коллекций

Создание индексов выполняется по следующим коллекциям:
- **users** (уникальный индекс по полю `username`)
- **notes** (составной индекс по полям `user_id`, `isArchived`, `created`)

Благодаря этому приложение может быть запущено на новой пустой (без коллекций) базе данных без каких-либо дополнительных действий.

## Справочник по API

### Авторизация:

| Метод | URL | Описание |
|-------|-----|----------|
| POST | /signup | Регистрация |
| POST | /login | Авторизация |
| GET | /logout | Выход из аккаунта |

### Работа с заметками:

| Метод | URL | Описание |
|-------|-----|----------|
| GET | /api/notes | Получение списка заметок |
| GET | /api/notes/:id | Получение отдельной заметки |
| GET | /api/notes/:id/pdf | Экспорт заметки в формате PDF |
| POST | /api/notes | Создание заметки |
| PATCH | /api/notes/:id | Обновление заметки |
| DELETE | /api/notes/:id | Удаление отдельной заметки |
| DELETE | /api/notes | Удаление всех заметок из архива |
